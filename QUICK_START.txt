DATE & TIME FORMAT GUIDE
========================

DATES:
  Format: DD-MM-YYYY
  Examples:
    03-02-2026  (3rd February 2026)
    15-01-2025  (15th January 2025)
    31-12-2024  (31st December 2024)
    01-03-2026  (1st March 2026)

AGE:
  Format: Number + Unit
  Examples:
    25 years    (standard)
    5 years     (pediatric)
    60 years    (elderly)
    Or: 25 Yrs, 5 Yrs

REPORT TIMESTAMP:
  Automatically added to PDF
  Format: YYYY-MM-DD HH:MM:SS
  Example: 2026-02-03 14:32:15

================================================================================
TROUBLESHOOTING
===============

PROBLEM: Application won't start
SOLUTION:
  • Verify Python is installed: python --version
  • Run as Administrator if needed
  • Check g:\dr_vinita\xml convert folder exists

PROBLEM: "Required fields" error
SOLUTION:
  • Fill in ALL fields marked with asterisk (*)
  • Minimum required:
    - Report ID
    - Patient Name
    - Age
    - Gender
    - Receipt Date

PROBLEM: Can't save PDF file
SOLUTION:
  • Choose a different folder
  • Check folder permissions
  • Verify disk space available
  • Try saving to My Documents

PROBLEM: Text appears cut off
SOLUTION:
  • Use shorter text entries
  • Break long text across multiple lines
  • Avoid extremely long paragraphs

PROBLEM: Special characters not showing
SOLUTION:
  • Use standard letters and numbers
  • Avoid symbols: @#$%^&*
  • Use parentheses: () for grouping
